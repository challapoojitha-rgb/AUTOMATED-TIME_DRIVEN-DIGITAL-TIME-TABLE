# Automated Time-Driven Digital Notice Board System (Java Version)

##  Project Overview

The **Automated Time-Driven Digital Notice Board System** is a console-based application developed using **Java** to manage and display notices digitally.
The system allows users to create, view, and manage notices with details such as title, description, priority level, channel, and expiry date.

The program automatically organizes notices based on priority and removes expired notices to ensure that only relevant information is displayed.

---

##  Features

* Add new notices with title, description, priority, expiry date, and channel
* View all active notices
* Delete notices from the system
* Automatically remove expired notices
* Sort notices based on priority (High → Medium → Low)
* Simple console-based menu interface

## ▶ How to Run the Program

### 1. Save the Program

Save the Java file as:

```
DigitalNoticeBoard.java
```

---

### 2. Open Command Prompt / Terminal

Navigate to the folder where the file is saved.

Example:

```
cd Documents
```

---

### 3. Compile the Program

```
javac DigitalNoticeBoard.java
```

This will create:

```
DigitalNoticeBoard.class
```

---

### 4. Run the Program

```
java DigitalNoticeBoard
```
